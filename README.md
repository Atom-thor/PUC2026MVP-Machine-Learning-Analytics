# MVP Sprint Machine Learning - 1° Semestre 2026

O repositório tem como objetivo o armazenamento dos datasets e notebook do Google Collab utilizados para o MVP da disciplina de Machine Learning & Analytics.

Conteúdo:

1. MVP_ML_Analytics_20261_João_B_A_Morais.ipynb
    * Notebook principal do trabalho. Para executar no Google Collab, basta abri-lo e clicar em ["Open in Collab"](https://colab.research.google.com/github/Atom-thor/PUC2026MVP-Machine-Learning-Analytics/blob/main/MVP_ML_Analytics_20261_Jo%C3%A3o_B_A_Morais.ipynb)
2. arrhythmia.data
    * Base de dados utilizada para o MVP.
3. arrhythmia.names
    * Documentação do dataset conforme fornecida pelo autor original (Guvenir _et al._, 1997)
